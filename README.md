## 🚀 Quick Start

## 1. Clone the Repository
```bash
git clone https://github.com/diliprx/portfolio-source.git
cd portfolio-source
```
## 2. Install Dependencies

```bash
npm install
```

## 3. Run Development Server
```
npm run dev
```
## Open to

``` bash
http://localhost:5173
```
## Build for Production

```
npm run build
```

## 4. Deployment

## 🔹 Netlify (Recommended)
## Option 1: CLI
````
npm install -g netlify-cli
npm run build
netlify deploy --dir=dist --prod
````

## Option 2: Drag & Drop
```
Run npm run build
Go to https://netlify.com
Drag and drop the dist/ folder
```
## 🐳 Docker Deployment (Optional)
```
docker-compose up -d --build
```
